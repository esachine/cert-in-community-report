CIVN-2026-0365 Vulnerability Note July 16, 2026

Multiple Vulnerabilities in Mozilla Firefox

Severity: Critical

What happened

Multiple vulnerabilities have been reported in Mozilla Firefox which could allow a remote attacker to execute arbitrary code, bypass security restrictions, denial of service, and data manipulation on the targeted system.

Software affected

Mozilla Firefox versions prior to 152.0.6.

Who is affected

All end-user organizations and individuals using Mozilla Firefox.

Risk

High risk of denial of service (DoS), and data manipulation on the targeted system.

Potential impact

Potential for remote code execution and bypass security restrictions.

Details

Mozilla Firefox is a free and open-source web browser developed by the Mozilla Foundation. Multiple vulnerabilities exist in Mozilla Firefox due to an Invalid pointer in the JavaScript of WebAssembly component and Improper site isolation in the DOM of Navigation component. Successful exploitation of these vulnerabilities could allow a remote attacker to execute arbitrary code, bypass security restrictions, denial of service, and data manipulation on the targeted system.

What to do

Apply the appropriate security update provided by the vendor: mfsa2026-67

References

  • www.mozilla.org/en-US/security/advisories/mfsa2026-67/

CIVN-2026-0364 Vulnerability Note July 15, 2026

Remote Code Execution Vulnerability in SP Pagebuilder Extension for Joomla

Severity: High

What happened

A vulnerability has been reported in the Joomshaper SP Page Builder extension for Joomla, which could allow an unauthenticated attacker to execute arbitrary code, or completely compromise the affected website.

Software affected

Joomshaper SP Pagebuilder extension versions prior to 6.6.2

Who is affected

All end-user organizations and individuals using the SP Page Builder extension for Joomla.

Risk

Risk of remote code execution.

Potential impact

Unauthorized access to sensitive data, data manipulation and complete system compromise.

Details

SP Page Builder is a page builder extension for Joomla that enables users to create responsive web pages using a drag-and-drop interface without extensive coding. A vulnerability has been reported in the SP Page Builder extension for Joomla due to improper validation of filetype during before upload. An attacker could exploit this vulnerability by uploading a malicious file. Successful exploitation of this vulnerability could allow an attacker to execute arbitrary code, or completely compromise the affected website.

What to do

Update to the latest version of the extension: page-builder

References

  • www.joomshaper.com/page-builder/

CIVN-2026-0363 Vulnerability Note July 13, 2026

Multiple Vulnerabilities in Ivanti Sentry

Severity: Critical

What happened

Multiple vulnerabilities have been reported in Ivanti Sentry, which could allow a remote unauthenticated attacker to execute arbitrary code with root privileges, bypass authentication, create arbitrary administrative accounts, and gain full administrative access on the targeted system.

Software affected

Ivanti Sentry 10.5.x versions prior to 10.5.2 Ivanti Sentry 10.6.x versions prior to 10.6.2 Ivanti Sentry 10.7.x versions prior to 10.7.1

Who is affected

IT administrators, security administrators, network administrators, and organizations using Ivanti Sentry.

Risk

Critical risk of complete system compromise, unauthorized administrative access, and execution of arbitrary commands.

Potential impact

Potential for complete system compromise, unauthorized administrative access, and arbitrary code execution with root privileges.

Details

Ivanti Sentry, formerly known as MobileIron Sentry, is an in-line gateway that manages, encrypts, and secures traffic between mobile devices and backend enterprise systems. Multiple critical vulnerabilities have been reported in Ivanti Sentry due to improper neutralization of special elements used in an operating system command and authentication bypass using an alternate path or channel. Successful exploitation of these vulnerabilities could allow a remote unauthenticated attacker to execute arbitrary code with root privileges, bypass authentication, create arbitrary administrative accounts, and gain full administrative access on the targeted system. Note: CVE-2026-10520 is being actively exploited in the wild.

What to do

Upgrade to Ivanti Sentry versions 10.5.2, 10.6.2 or 10.7.1, as applicable, by applying the security updates provided by Ivanti immediately: hub.ivanti.com

References

  • hub.ivanti.com/s/article/Security-Advisory-Ivanti-Sentry-CVE-2026-10520-CVE-2026-10523?language=en_US
